This project represents the transition of my professional portfolio from a proprietary platform (Adobe Portfolio) to a custom-built, static site generated with **Hugo**.

### The Motivation
Adobe Portfolio provided a convenient starting point, but I reached the limits of its customization. As a Technical Director and Professor, I wanted a site that was:
1.  **Portable**: Ownership over the source code and content.
2.  **Fast**: Optimized static delivery.
3.  **Unique**: A design tailored to my specific "Six Bars" visual hierarchy.

### The Process
The entire site is being built by reverse engineering my previous layout and refining it through an iterative pairing process with **Antigravity**, an advanced AI coding assistant. 

We are currently translating the visual language of the original site into clean **HTML**, **Vanilla CSS**, and **Tailwind utility classes**, ensuring that the resulting code is as professional as the work it showcases.

### Tools Used
*   **Hugo**: For high-performance static site generation.
*   **Tailwind CSS**: For fine-grained utility styling.
*   **Antigravity (AI)**: For collaborative development and procedural content generation.
